## Integration

Set up a paymaster for the smart account by providing only the paymaster API key. The default mode used is `Sponsored` mode in this case.
:::info

[Click here](/dashboard/paymaster) to learn more about our dashboard and how to get your Paymaster api key.

:::

```typescript
  import { BiconomySmartAccountV2 } from "@biconomy/account";

  const biconomySmartAccount = await BiconomySmartAccountV2.create(
    {
      signer: signer,
      chainId: 80001
      biconomyPaymasterApiKey: "https://docs.biconomy.io/dashboard/paymaster"
      bundlerUrl: "bundler url"
    }
  );
```

If you wish to pass "strictMode" to your Paymaster you need to create the instace.

```typescript
  import { IPaymaster, BiconomyPaymaster } from "@biconomy/paymaster";
  import { BiconomySmartAccountV2 } from "@biconomy/account";

  const paymaster: IPaymaster = new BiconomyPaymaster({
    //https://dashboard.biconomy.io/ get paymaster urls from your dashboard
    paymasterUrl: "",
    strictMode: true
  });

  const biconomySmartAccount = await BiconomySmartAccountV2.create(
    {
      signer: signer,
      chainId: 80001
      paymaster,
      bundlerUrl: "bundler url"
    }
  );
```

You have now assigned the Paymaster to the smart account. See our [tutorials](/tutorials) for in depth integrations of the Smart Account and Paymaster.
